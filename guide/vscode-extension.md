# VSCode Extension

The ICE VSCode extension provides an integrated development experience for Internet Computer projects using ICE.

## Installation

1. Open VSCode
2. Go to Extensions (`Cmd+Shift+X` / `Ctrl+Shift+X`)
3. Search for: ICE Extension
4. Click Install
